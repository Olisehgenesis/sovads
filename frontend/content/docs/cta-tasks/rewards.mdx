---
title: Rewards & limits
description: SovPoint and G$ rewards, plus per-task caps.
---

| Field | Meaning |
| --- | --- |
| `rewardPoints` | SovPoints awarded per completion. No minimum. |
| `rewardGs` | Optional G$ payout. Minimum **6 G$** when set. |
| `budgetGs` | Optional task-level cap (otherwise the campaign budget applies). |
| `maxPerWallet` | Hard cap of completions per wallet. |
| `cooldownSecs` | Minimum seconds between completions for the same wallet. |
| `startDate` / `endDate` | Optional ISO 8601 window when the task is live. |
