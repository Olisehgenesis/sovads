---
title: Install
description: Install the SovAds SDK from npm, pnpm, or via a CDN script tag.
---

```bash title="npm"
npm install sovads-sdk
```

```bash title="pnpm"
pnpm add sovads-sdk
```

```html title="CDN (no build step)"
<script type="module" src="https://ads.sovseas.xyz/_sovads_sdk/index.js"></script>
```
